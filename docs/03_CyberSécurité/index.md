# Cybersécurité

La cybersécurité occupe une place essentielle dans le BTS SIO. Elle vise à protéger les systèmes, les données et les utilisateurs contre les menaces numériques.

## Notions étudiées
- Gestion des droits et permissions
- Sécurisation Windows et Linux
- Analyse réseau
- Chiffrement et authentification
- Surveillance et journalisation
