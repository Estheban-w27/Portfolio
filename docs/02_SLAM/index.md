# Spécialité SLAM

La spécialité SLAM (Solutions Logicielles et Applications Métiers) se concentre sur le développement d’applications et la gestion de projets logiciels. Elle permet d’apprendre à concevoir, coder, tester et maintenir des solutions adaptées aux besoins des organisations.

## Compétences développées
- Développement d’applications web et logicielles
- Gestion de bases de données
- Analyse des besoins et modélisation
- Travail collaboratif avec Git
- Tests et maintenance applicative
